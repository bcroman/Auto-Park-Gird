Auto Park Gird

Learn CSS Grid through Interactive Challeneges

Auto Prk Grid is an interactive learning tool which is designed to tech the concept of CSS Grid through a range of levels.
Progress through a series of levels, learning new concepts and applying them by writing CSS code to position cars and vans (Grid Items) on a parking lot (Grid).
Optional hints available to help solve the challenges and get useful feedback for incorrect answers.
