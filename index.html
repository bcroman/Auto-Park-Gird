Auto Park Grid

Learn CSS Grid through Interactive Challenges

Auto Park Grid is an interactive learning tool which is designed to teach the concept of CSS Grid through a range of levels. Progress through a series of levels, learning new concepts and applying them by writing CSS code to position cars and vans (Grid Items) on a parking lot (Grid).
Optional hints available to help solve the challenges and get useful feedback for incorrect answers.
