About Auto Park Grid

This website is gamified learning tool which is designed to help beginners developers and learn and understand CSS Grid through interactive, puzzle based challenges. Instead of memorising concepts, users learn by writing real CSS and seeing the update visually.

How it works

Each level introduces a specific CSS Grid concept such as rows, columns, and spanning.

  1. Read the learning content and challenge details.
  2. Write CSS into the editor to control the grid layout.
  3. Check your solution against the challenge target.
  4. Receive feedback and hints if needed.
  5. Progress to the next level and learn a new concept.

Features

Solutions are validated by checking the final layout output, rather than matching text base. This means there are multiple correct solutions can be accepted, as long the resulting render matches the level goal.
