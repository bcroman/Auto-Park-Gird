About Auto Park Grid

This website is gamified learning tool which is designed to help beginners developers and learn and understand CSS Gird through interactive, puzzle based challenges.
Instead of memorising concepts, users learn by writing real CSS and seeing the update visually.

Each introduces a specific CSS Grid concept such are: rows and columns and apanning.
Each level will follow the same progress:
1. Read the Learning Content and Challenge Details
2. Write CSS into the editor to control the grid layout.
3. Check your solution to challenge target.
4. Receive feedback on your solution and hints if needed.
5. Progress to the next level and learn a new concept.

Solutions are validated by checking the final lauout output, rather than matching text base.
This means there are multipe correct solutions can be accepted, as long the resulting render matchs the level goal.
